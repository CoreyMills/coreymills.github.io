Search for a Star 2021

A game created for "Search for a Star 2021" using the Unity engine.

Chosen Demise Submission

Chosen Demise is an arena survival game where you choose between sets of upgrades which apply to both the player and the opponents.

This game was made under 80 hours as part of the challenge and was created with the theme of "choice" in mind.

For the project I used AI, Navmesh Pathfinding, and Rigidbody Physics.
